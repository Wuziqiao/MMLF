# MMLF
This is the source code for paper "MMLF: Multi-Metric Latent Factor Analysis for High-Dimensional and Sparse Matrices"

1.Run our algorithm

(1) Download the file “MMLF.rar”.

(2) Unzip the file " MMLF.rar".

(3) Import files into a JAVA Integrated Development Environment, like “Eclipse”, refer to https://www.eclipse.org/downloads/.

(4) Open “Main_Function.java” and run.
